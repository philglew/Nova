The Nova API


Our REST API is not like other HCM platform APIs. It's much better. Here's why...


Each module is designed to optimise your HR processes, allowing your business to reach new heights with minimal effort.

Unrivalled security


Our API is built with security at the very forefront. We use the latest encryption technologies to ensure your data is safe, both at rest and in transit.


Most HCM platforms offer an API which is accessed via the Odata standard. So does Nova; but where we go further to ensure your data is protected, is by using assymetric cryptography. This means that your data is encrypted using a public key, and can only be decrypted using a private key. This ensures that your data is safe, so you don't need to worry about API keys or passwords finding their way into the wrong hands.

Full data access


Full data access means exactly that. All of your data is available via our API; no exceptions. You, and only you, are free to interrogate your data in any way you see fit. You can use the data to build custom reports, or to integrate with other systems. The choice is yours.

Unlimited requests


Unlike other HCM platforms, we don't limit the number of requests you can make to our API. You can make as many requests as you like, whenever you like. We don't believe in charging you for the amount of times you access your own data.

Full documentation


Our API documentation is comprehensive and easy to follow. We provide code examples, and we're always on hand to help you with any questions you may have.

Our supported platforms


Our API is designed to work with any platform. Whether you're using a Windows, Mac, or Linux machine, our API will work for you. We also provide SDKs for popular programming languages, such as Python, Java, and C#.

Connectors


We have plug-and-play connectors available for a host of automation, data analysis and integration services including Power Automate, Power BI, Tableau and, of course, Postman.
